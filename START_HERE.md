## 💡 COMMON QUESTIONS ANSWERED

### "How long until I get interviews?"
**2-4 weeks** for first invitations. With 50 apps/day, you'll have 350+ applications in 1 week, 1,400+ in month 1.

### "How much does it really cost?"
**$0-5/month** using free tiers (Groq, Voyage, Supabase). Upgrade to $70-80/month for better AI and higher limits.

### "Is this legal?"
**Yes**, but check each platform's Terms of Service. LinkedIn is gray area (use cautiously). Indeed generally allows automation. Always apply honestly.

### "What if I get banned?"
- Start slow (10-20 apps/day)
- Use rate limiting
- Rotate sessions
- Add human-like delays
- Don't be too aggressive

### "Do I need coding skills?"
**No** for basic use. Setup is automated. **Yes** for customization (but well documented).

### "Can I use this for non-tech jobs?"
**Yes!** Just update `target_roles` in config. System works for any industry.

### "How do I stop it?"
```bash
# Local: Ctrl+C
# Cloud: railway down (or docker-compose down)
```
